# SauceDemo Cypress Automation

## Description

Automation framework developed with:

- Cypress
- Cucumber (Gherkin)
- JavaScript
- Page Object Model

## Funcionalidades automatizadas

### Inicio de sesión
- ✅ Login exitoso
- ✅ Usuario bloqueado
- ✅ Validación de contraseña incorrecta
- ✅ Validación de usuario inexistente
- ✅ Validación de usuario vacío
- ✅ Validación de contraseña vacía
- ✅ Validación de usuario y contraseña vacíos
- ✅ Validación de espacios en blanco
