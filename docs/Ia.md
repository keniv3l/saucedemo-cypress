# Uso de IA durante el proyecto

## Codex

Se utilizó para:

- Autocompletado.
- Refactorización.
- Generación inicial de Step Definitions.

Todo el código fue revisado y adaptado manualmente.

## Playwright MCP

Se utilizó para:

- Explorar la aplicación.
- Identificar funcionalidades.
- Proponer escenarios BDD.
- Verificar cambios en la interfaz.

No se utilizó para generar la automatización final.
